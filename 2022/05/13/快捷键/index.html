快捷键

Created At : 2022-05-13 16:36
Count:233
适用 快捷键
visual studio，typora ctrl+f 查找
cmd cd /d file_place 跳转到指定文件夹位置
文件夹 鼠标中键“在CMD中打开”进入cmd端口
typora shift+enter 表格中换行
excel ctrl + T：普通表格变成超级表
windows win+R，cmd进入命令面板，regedit进入注册表
sublime ctrl+shift+L 多行代码编辑
GitHub网站快捷操作
找项目：按S直接定位到搜索框，用高级搜索Advanced search
查看项目文件：T键，激活“文件查找器”；L键，跳转到代码指定行；B键查看代码修改记录
ctrl+k，打开命令面板
”。“句号键，打开在线vscode编辑器
在线运行项目：在项目地址前加上gitpod.io/#/前缀，例如Dashboard — Gitpod
把项目开头网址改成https://github1s.com/开头
快速下载GitHub项目文件，在本地路径：git clone https://github.com/项目文件网址.git
